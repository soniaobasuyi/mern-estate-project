# Real Estate Listing Web App

This application was built to demonstrate a full-stack architecture using the MERN stack.
It enables users to view property listings, manage data through CRUD operations, and interact with a responsive user interface optimized for both desktop and mobile devices.

The project focuses on building scalable components, implementing secure user interactions, and creating a clean, modular code structure.

Live Demo: https://sonia-estate-project.onrender.com/

## Tech Stack
### Frontend
- React  
- SCSS
- Responsive CSS layout
- React Router for navigation

### Backend
- Node.js 
- Express.js 
- REST API architecture

### Database 
- MongoDB

### Deployment
- Render

## Features 
- Browse available real estate listings 
- View property details 
- Create new property listings 
- Edit and update listings 
- Delete listings 
- Responsive layout for mobile and desktop 
- Dynamic routing for property pages
- Sign-up, sign-in and delete account

## Application Architecture

The project follows a typical MERN stack architecture:

Client (React) → API (Node.js / Express) → Database (MongoDB)

Frontend components communicate with backend REST APIs to fetch and manipulate property data stored in MongoDB.

The frontend uses reusable components and modular styling to ensure maintainability and scalability.

## Installation (Run Locally)
1. Clone repository
```sh
git clone https://github.com/soniaobasuyi/mern-estate-project.git
```

2. cd project folder 
3. Install backend dependencies and start server
```sh
npm install
npm run dev
```
 
4. In a new terminal, install frontend dependencies and start the client 
```sh
cd frontend
npm install
npm run dev
```

Backend runs on port 3000

Frontend runs on port 5173
